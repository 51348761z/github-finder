# GitHub Finder

A lightweight application to search for GitHub users, built as a learning project to practice modern frontend development techniques.

## Features

- Search for GitHub users by username
- View user details and profile information
- Recent search history
- Data caching and state management

## Tech Stack

- **Framework**: React + TypeScript
- **Build Tool**: Vite
- **Data Fetching**: TanStack Query v5
- **Utilities**: `use-debounce`
